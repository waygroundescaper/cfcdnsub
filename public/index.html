Subscription Generator (China Supported)

Thank you for using ShadowEscaper's Subscription Generator! This tool is designed to help you easily replace the server addresses in your self-hosted vmess/vless/trojan nodes with your preferred IPs, and export subscription links compatible with Clash, V2rayN, Shadowrocket, Surge, and more. Please note that this tool is intended for users in China and may not work properly outside of China due to network restrictions.

What is this project?

ShadowEscaper is a proxy service that helps you stay hidden on the internet and protects your online safety.

Mainly compatible with common WS + TLS formats of self-hosted VMess / VLESS / Trojan nodes.

Supports IP, IP:port, domain, domain:port, with optional comments after #.

Notes

  • This tool does not find preferred IPs for you — it only replaces existing nodes with the preferred IPs you provide.
  • The generated links contain the same node content in different export formats, each suited for different clients.
  • It's recommended to keep "Keep Original Host / SNI" enabled, especially for Cloudflare CDN scenarios.
  • Node information in subscription links is encrypted with a server-side key. Please do not share these links publicly.

How to Use with Clients

  • V2rayN: Use the raw subscription link
  • Shadowrocket: Use the raw subscription link
  • Clash / Mihomo: Use the Clash subscription link
  • Surge: Use the Surge subscription link
